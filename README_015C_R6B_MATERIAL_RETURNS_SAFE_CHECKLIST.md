# CLONEXA 015C-R6B — Material Returns Safe Checklist

Corrige devolución de Materiales sin tocar Web App ni Inventario.

- Número de orden funciona como buscador.
- Botón Devolución precarga checklist por orden.
- Oculta devolución para filas sin orden.
- Backend agrega endpoints separados para búsqueda, checklist y devolución por unidades.
- Devolución por checkbox de unidades entregadas.
- Suma inventario únicamente por unidades seleccionadas.
